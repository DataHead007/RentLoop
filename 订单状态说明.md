# 订单状态说明

## 订单状态类型

订单有以下几种状态：

1. **pending（待确认）**
   - 含义：订单刚刚创建，等待确认
   - 操作：可以确认订单，开始执行

2. **confirmed（已确认）**
   - 含义：订单已确认，准备发货
   - 操作：可以标记为进行中，开始发货

3. **in_progress（进行中）**
   - 含义：订单正在执行中，设备已租出
   - 操作：可以执行收货操作（客户归还设备）

4. **completed（已完成）**
   - 含义：订单已完成，设备已收回
   - 操作：订单结束，不可再修改

5. **cancelled（已取消）**
   - 含义：订单已取消
   - 操作：订单结束

## 状态流转

```
pending（待确认）
    ↓
confirmed（已确认）
    ↓
in_progress（进行中）
    ↓
completed（已完成）

或

任何状态 → cancelled（已取消）
```
