## 📋 接下来必须完成的 2 个步骤

### 步骤 1：在 Supabase 中创建数据库表 ⚠️ **重要**

1. 打开浏览器，访问：https://supabase.com/dashboard
2. 登录并选择你的项目
3. 点击左侧菜单的 **SQL Editor**
4. 点击 **New query** 按钮
5. 打开项目文件 `supabase/schema.sql`，**复制全部内容**
6. 粘贴到 SQL Editor 中
7. 点击 **Run** 按钮（或按 Cmd/Ctrl + Enter）
8. 等待执行完成，应该显示 ✅ "Success. No rows returned"

**这一步是必须的，否则应用无法正常工作！**

### 步骤 2：创建 Storage Bucket ⚠️ **重要**

1. 在 Supabase Dashboard 左侧，点击 **Storage**
2. 点击 **New bucket** 按钮
3. 填写信息：
   - **Name**: `serial-numbers`（必须完全一致）
   - **Public bucket**: ✅ **勾选**（这样可以直接访问图片）
4. 点击 **Create bucket**

**这一步也是必须的，否则无法上传序列号照片！**

## 🚀 完成上述步骤后，启动应用

## ⚠️ 如果遇到问题

### API Key 错误

如果你提供的 "Publishable API Key" 不工作，请：

1. 进入 Supabase Dashboard → Settings → API
2. 找到 **Project API keys** → **anon public** key
3. 复制这个 key
4. 编辑 `.env.local` 文件，替换 `NEXT_PUBLIC_SUPABASE_ANON_KEY` 的值

### 检查配置是否正确

# 查看环境变量配置
