# 发票上传功能说明

## 功能概述

为设备管理添加了购买发票上传功能，方便管理设备的购买凭证和文档。

## Supabase Storage 设置

### 1. 创建 Storage Bucket

需要在 Supabase 中创建一个名为 `invoices` 的 Storage bucket：

1. 登录 Supabase Dashboard
2. 进入你的 RentLoop 项目
3. 点击左侧菜单的 **"Storage"**
4. 点击 **"New bucket"** 创建新桶
5. 设置：
   - **Name**: `invoices`
   - **Public bucket**: ✅ 勾选（公开访问）
6. 点击 **"Create bucket"** 创建

### 2. 设置存储策略（可选）

如果需要更安全的访问控制，可以设置存储策略，但对于公开桶，默认所有用户都可以读取。

## 功能特点

1. **支持的文件格式**：
   - 图片：JPEG、JPG、PNG、WebP
   - 文档：PDF
   - 文件大小限制：最大 10MB

2. **上传体验**：
   - 拖拽上传界面
   - 上传进度提示
   - 错误提示（文件类型、大小限制）

3. **文件管理**：
   - 可以查看已上传的发票
   - 可以删除已上传的发票
   - 在新窗口打开查看

4. **显示位置**：
   - 创建/编辑设备表单中显示
   - 设备详情页中显示发票链接

## 使用方法

### 上传发票

1. 在创建或编辑设备时，找到"购买信息"部分
2. 在"购买发票"字段中：
   - 点击上传区域
   - 选择文件（图片或 PDF）
   - 等待上传完成

3. 上传成功后：
   - 显示"发票已上传"提示
   - 可以点击"查看发票"在新窗口打开
   - 可以点击 ❌ 按钮删除发票

### 查看发票

- 在设备详情页的"基本信息"卡片中
- 点击"查看发票"链接即可在新窗口打开

## 注意事项

- 文件大小限制为 10MB
- 只支持图片（JPEG、PNG、WebP）和 PDF 文件
- 上传的发票会存储在 Supabase Storage 中
- 发票 URL 会保存在设备的 `purchase_invoice_url` 字段中
